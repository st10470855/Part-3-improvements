Young Brazil Football Club Website Project 

Project Overview 

This project is a responsive website for the Young Brazil Football Club, designed to showcase the club's mission, services, news, and provide a contact/registration form. The website is intended for parents, players, and the community, emphasizing youth development, training, and engagement. 

 

Pages 

The website consists of the following pages: 

Home Page (home.html) 

Features the club's logo, navigation menu (with a hamburger menu for mobile), and a hero section. 

Interactive image section: pictures of the team, training sessions, and matches that highlight when hovered over, giving users a visual interaction. 

Embedded video of the club activities placed below the images for better flow. 

Improved CSS styling with consistent green and yellow theme across all pages. 

Meta tags for description, author, and keywords included for SEO. 

About Us (about.html) 

Provides an overview of the club, mission, and philosophy. 

Emphasis on nurturing children's growth both on and off the field. 

Text content formatted for readability with headings, paragraphs, and highlights. 

Products & Services (products.html) 

Lists professional football training programs, youth development academy, and event hosting. 

Note: The club is located in a rural area, so merchandising items (jerseys, scarves, etc.) are planned for the future but currently unavailable. 

News Update (news.html) 

Lists latest club news and events, e.g., tournament wins, new training camps. 

Articles are clearly separated with headings and paragraphs. 

Contact Us (contact.html) 

Includes a registration form with fields: Full Name, Email, Phone, Age, Message. 

Integrated EmailJS for sending form submissions directly via email. 

SDK included via <script src="https://cdn.jsdelivr.net/npm/emailjs-com@2/dist/email.min.js"></script>.
